# BraxenAI

## 🚀 Run Locally

**Prerequisites:** Node.js

1. Install dependencies

npm install

2. Add your Gemini API key in `.env.local`

GEMINI_API_KEY=your_api_key_here

3. Start the development server

npm run dev
